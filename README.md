## Materials Used
* Raspberry Pi
* Breadboard
* 1 resistor (220 ohm)
* 3 LEDs (1 red, 1 yellow. 1 green)
* 1 push button
* 5 male-to-female jumper wires
* 3 male to male jumper wires
